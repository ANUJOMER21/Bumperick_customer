Quick Start Guide

Get up and running in minutes

1

Clone Repository

Get the source code from your repository

git clone <repository-url>
cd Bumperick
2

Configure API Keys

Set up your Google Maps API key and Firebase configuration

  • Add Google Maps API key in AndroidManifest.xml
  • Place google-services.json in app directory
  • Update base URL in AppModule.kt if needed
3

Build Project

Sync dependencies and build the application

./gradlew build
4

Run Application

Deploy to your device or emulator

./gradlew installDebug
